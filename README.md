# Zoyo Bathware App

Zoyo Bathware is a Flutter-based application designed to showcase bathroom products such as faucets, showers, wash basins, and accessories.
The app provides a clean and modern interface where users can explore product collections and view detailed product information.

The project demonstrates building a **cross-platform Flutter application** with modular architecture and reusable UI components.

---

# Features

### Product Showcase

Users can browse bathware products including:

* Faucets
* Showers
* Wash basins
* Bathroom accessories
* Sanitary products

Each product includes images and detailed descriptions.

---

### Cross Platform Support

The application is built using **Flutter**, allowing it to run on multiple platforms:

* Android
* iOS
* Web
* Windows
* macOS
* Linux

# Getting Started

### 1 Clone the repository

```
git clone https://github.com/nadeerep07/Zoyo_bathware_main.git
```

### 2 Install dependencies

```
flutter pub get
```

### 3 Run the application

```
flutter run
```

For web:

```
flutter run -d chrome
```
